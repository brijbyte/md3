<Demo of="primary" title="Primary tabs">
  The active indicator hugs the label and stretches elastically between tabs, per MDC Android. Arrow
  keys move focus and activate.
</Demo>

<Demo of="secondary" title="Secondary tabs">
  A flatter 2dp indicator spans the full tab width — for a second level of content within a page.
</Demo>

<Demo of="icons" title="With icons">
  Primary tabs stack the icon above the label (64dp); secondary tabs keep it inline.
</Demo>

<Demo of="badges" title="With badges">
  Pass `badge` to a Tab: it anchors to the icon's top-right corner, or sits inline after the label
  on text-only tabs. Badges are visual — put the count in the tab's `aria-label` too.
</Demo>
