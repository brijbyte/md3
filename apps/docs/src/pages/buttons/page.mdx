## Button sizes

Five sizes (`xsmall`–`xlarge`) per the expressive spec; `small` is the default. Height, padding,
icon size, typescale, and outline width all scale together.

<ButtonSizes />

## Shapes & toggle

Round or `shape="square"`, morphing to a smaller radius while pressed. Toggle buttons (`toggle`,
Base UI Toggle underneath) flip shape and color when selected; there is no text-variant toggle.

<ButtonShapes />

## Floating action buttons

Also called FABs, these are the only buttons that float above the surface.

```tsx
import { Fab } from "@brijbyte/md3-react/fab";
```

```css
@import "@brijbyte/md3-react/button.css";
@import "@brijbyte/md3-react/fab.css";
```

<SpecLinks
  links={[
    {
      label: "MD3 spec",
      href: "https://m3.material.io/components/floating-action-button/overview",
    },
  ]}
/>

Baseline 56dp plus the expressive `medium` (80dp) and `large` (96dp) sizes; a `label` renders
the extended FAB at any size. Colors come in two families: soft `*-container` (default) and
bold tone roles. The 40dp small FAB and surface colors are deprecated by the expressive update.

<Fabs />

### Positioning

`Fab` only renders the button — it does not float by itself. Position it yourself, typically
`position: fixed` in the bottom-right corner with the spec's 16dp margin from the screen edges
(pass a `className` or `style`). Since the FAB covers whatever scrolls beneath it, also give
the content enough bottom padding (FAB height + margins, ~88px for the baseline size) so the
last elements can scroll up past it. Here the FAB is `position: absolute` inside the mock
screen so it stays put while the content scrolls.

<FabPositioning />

### Standard

Related buttons with 12dp spacing; each keeps its own shape. Press and hold one — it grows 15%
wider while its neighbors compress to absorb the change.

<ButtonGroupStandard />

### Connected

Buttons fuse into a pill: 2dp gaps, 8dp inner corners, full outer corners. Selected toggles round
fully; press-and-hold a toggle to see the inner corners tighten.

<ButtonGroupConnected />
