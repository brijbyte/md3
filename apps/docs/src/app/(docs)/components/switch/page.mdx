## States

A switch commits its choice the instant it's toggled, so use it for a single binary
setting that takes effect immediately — on or off, never one of several options — rather
than one a form submits later. For choosing among several options, reach for
[Radio](/components/radio) instead. It renders on Base UI's Switch primitive, giving you
correct keyboard and screen-reader behavior out of the box, and every disabled/hover/
focus/pressed combination below is styled to spec.
